Build an application description: Build an application using your favorite programming language. headcontent: Use your favorite programming language to build an application that uses YSQL or YCQL APIs. image: /images/section_icons/develop/api-icon.png block_indexing: true menu: v2.1: identifier: build-apps parent: quick-start weight: 150 --- An application interacts with YugabyteDB using a client library, also known as a client driver. Client drivers handle all interaction with the database in a language appropriate to the application. Here is a list of client drivers in various languages for YugabyteDB for building transactional, internet-scale applications. Note that the YugabyteDB YSQL and YCQL APIs are wire-compatible with PostgreSQL and Cassandra Query Language, respectively. As a result, you will be leveraging the client drivers of these popular languages to connect to YugabyteDB.

We will also be integrating YugabyteDB with the most popular ORM frameworks for each of these languages.
