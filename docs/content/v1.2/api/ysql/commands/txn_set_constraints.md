## Synopsis

`SET CONSTRAINTS` command checks timing for current transaction.

### Grammar
```
set_constraints ::= SET CONSTRAINTS { ALL | name [, ...] } { DEFERRED | IMMEDIATE }
```

## Semantics

- Attributes in `SET CONSTRAINTS` does not apply to `NOT NULL` and `CHECK` constraints.
