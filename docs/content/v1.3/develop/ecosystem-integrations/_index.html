--- title: Ecosystem integrations linkTitle: Ecosystem integrations description: Ecosystem integrations headcontent: YugabyteDB is driver compatible with Apache Cassandra, Redis and PostgreSQL (beta) clients. It can therefore work with the ecosytem of projects built around these APIs. This page lists some of the ecosystems that YugabyteDB can work with. image: /images/section_icons/develop/ecosystem-integrations.png block_indexing: true menu: v1.3: identifier: ecosystem-integrations parent: develop weight: 570 ---
Apache Kafka
Distributed streaming platform integrated with YCQL API using Kafka Connect.
Apache Spark
Powerful analytics processing engine integrated with the YCQL API.
JanusGraph
Scalable graph API that uses Cassandra-compatible YugabyteDB as it's underlying distributed database.
KairosDB
Fast time series metrics API that uses Cassandra-compatible YugabyteDB as it's underlying distributed database.
Presto
Ad-hoc interactive SQL queries on top of YugabyteDB.
Metabase
Ad-hoc interactive SQL queries on top of PostgreSQL-compatible YugabyteDB.
